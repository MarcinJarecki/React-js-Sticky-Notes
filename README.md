####Requirements:
- Web browser: Chrome, Firefox 
- Text editor: Atom, Sublime, Visual Studio Code etc.
- node.js v6

####Project structure
- /app - aplication file
- /dist - folder fo production version (minification)
- /eslin-rules - rules from /facebook/React
- /.vscode - Visual studio code configuration
